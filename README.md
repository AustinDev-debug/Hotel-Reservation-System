# Hotel Reservation System

A console-based hotel reservation system built in C#. This project allows users to manage hotel rooms through an interactive menu, making it easy to add rooms, reserve available rooms, check guests out, search for room information, and view every room currently in the hotel.

## Features

* Add new hotel rooms
* View all rooms
* Reserve available rooms
* Check guests out
* Search for a room by room number
* Input validation for both text and numeric values

## Concepts Used

* C#
* Object-Oriented Programming (OOP)
* Classes and Objects
* Constructors
* Multiple Class Files
* Lists (`List<T>`)
* Methods
* Loops (`for` and `foreach`)
* Switch Statements
* Input Validation

## What I Learned

This project helped me learn how constructors simplify object creation by allowing data to be assigned when an object is created. I also gained more experience working with lists of custom objects, updating object data, searching through collections, and organizing larger C# projects into separate class files.

## How to Run

1. Open the solution in Visual Studio.
2. Build the project.
3. Press **F5** or click **Start** to launch the application.
4. Follow the on-screen menu to manage hotel rooms.
